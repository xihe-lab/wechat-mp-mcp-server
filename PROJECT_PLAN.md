# 微信公众号 MCP 服务器 - 项目计划

## 项目目标

构建一个符合 MCP 标准的微信公众号服务器，实现 AI 驱动的自动发布流程。

## Sprint 1（2026-06-01 ~ 2026-06-14）

### 交付物

- 17 个 MCP 工具接口（认证 3 + 素材 2 + 图文 4 + 草稿 4 + 发布 4）
- SQLite 本地存储（Token/配置持久化）
- 错误处理 + 日志脱敏
- 单元测试 + API 文档

### 完成标准

- [x] 17 个 MCP 工具接口全部实现
- [x] 完整自动发布流程（内容→草稿→发布）
- [ ] 单元测试覆盖率 ≥ 80%（当前 22 个用例，4 个测试文件）
- [x] API 文档
- [ ] Claude Desktop 集成指南
